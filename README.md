# AI Telegram Assistant для компании Центр Красок #1

## Описание

Это MVP Telegram-бота с AI-ассистентом для компании Центр Красок #1.

Бот работает в формате обычного чата: пользователь пишет вопрос, бот отвечает на основе подготовленной базы знаний о компании.

## Возможности

- Ответы на вопросы о компании
- Ответы об услугах, товарах, адресе, графике работы и контактах
- Работа без меню и обязательных команд
- Интеграция с Telegram Bot API
- Интеграция с Groq AI API
- Ответы на основе базы знаний
- Ограничение галлюцинаций
- Контекст диалога
- UX: typing status, приветствие, обработка нетекстовых сообщений

## Технологии

- Python
- aiogram
- Groq AI API
- OpenAI-compatible SDK
- python-dotenv
- Telegram Bot API
